# BlackOut
웹에서 플레이 가능한 어몽어스 스타일의 2D 멀티플레이 마피아 게임. (최소 2인 이상 플레이 가능)

게임 시작 후 일정 시간이 지나면 암전이 진행됩니다. 암전된 공간에서 돌아다니며 일부 가구에 숨겨진 아이템을 찾고, 조합대를 찾아 조합해 폭죽을 사용해 살인자를 찾아보세요.
암전 상태일 때는 플레이어들의 닉네임이 보이지 않습니다. 투표를 통해 살인자를 찾아내 죽이면 생존자들의 승리, 생존자들을 모두 죽이면 살인마의 승리입니다.

🎮 **[웹에서 바로 플레이하기](https://BlackOut-PJT.github.io/blckout/BlackOut/)**

## Tech Stack
- **Engine**: Unity 3.16.3
- **Language**: C#
- **Network**: Photon PUN2
- **Platform**: WebGL (Github Pages 배포)

## 🕹️ 게임 방법
게임 시작 후 닉네임 입력 -> 기존 방에  참여 또는 새로운 방을 생성(공개/비공개 선택) -> 로비에서 참여자 모두가 Ready 상태 후 방장에게만 플레이 버튼 활성화
* **이동**: `W`, `A`, `S`, `D` 또는 `방향키`
* **상호작용**: `E` (가구에 다가가 아이템 줍기, 조합대 사용, 벽난로에서 투표 시작)
* **생존자 승리조건**: 생존자들은 살인마를 투표를 통해 색출하여 승리
* **살인마 승리조건**: 투표나 살인을 통해 생존자를 모두 제거
